  <h1>Form Builder Application (Survey & Quiz System)</h1>
  <p>
    Build a web-based Form Builder application that allows users to create, share, and collect responses for surveys and quizzes, similar to Google Forms. </p>

  - ### Tech Stack
  <ol> 
       Frontend: TypeScript + React 

         Backend: Django Rest Framework

        Database: PostgreSQL

        Auth: Email + Password (JWT), Google Authentication

        Optional: Tailwind CSS, Prisma, REST API
  </ol>



### User Roles
- **Form Creator (Admin/User)**
- **Respondent (Public User)**



###  Authentication & User Management
- **User registration & login** – Secure account creation.
- **Secure authentication** – Implementation using **JWT (JSON Web Tokens)**.
- **User Dashboard** – A private landing page for users to manage their forms.

###  Form Builder (Create Form)
- **Metadata:** Define Form Title and Description.
- **Question Types:** Short/Long text, Multiple choice, Checkbox, and Dropdown.
- **Logic:** Mark questions as **Required**, edit, delete, or reorder questions.

### Quiz Mode (Optional but Recommended)
- **Evaluation:** Assign correct answers and marks per question.
- **Automation:** Auto-calculate scores and show results after submission.

###  Sharing & Access
- **Public Links:** Generate shareable URLs for anyone to respond.
- **Access Control:** Single submission toggle and manual form closing.

###  Response Management
- **Creator Insights:** View responses in a structured **Table Format**.
- **Export:** Download data as **CSV** (Bonus).

###  Non-Functional Requirements
- **UI/UX:** Clean, intuitive, and **Mobile Responsive** design.
- **Stability:** Robust error handling and basic API documentation (Swagger).

###  Bonus Features
- Form preview mode, duplicate form, and response analytics (charts).
